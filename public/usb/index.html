ðŸ—€ Built-in Modules

Although the codekitty module is meant to make a lot of common coding tasks on your robot easier, there are many built-in modules which you can also use.

To see a list of all the built-in modules you can use on your robot, use:

help("modules")

If you want to know more about a specific module (what functions it will provide), use:

help("module-name")

For example, if you want to know what functions are in the math module, use:

help("math")

ðŸ—€ Time and Sleep

One (very) common task you will do when coding is to have your program wait a little bit. One of the built-in modules, time, contains a function called sleep.

For example, to have your program wait for two and a half seconds, first import the sleep function:

from time import sleep

Then call sleep with an argument of 3 (for 3 seconds):

time.sleep(3)
